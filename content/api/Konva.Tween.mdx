# Tween

```javascript
new Konva.Tween(config)
```

Tween constructor.  Tweens enable you to animate a node between the current state and a new state.  You can play, pause, reverse, seek, reset, and finish tweens.  By default, tweens are animated using  a linear easing.  For more tweening options, check out [Konva.Easings](/docs/api/Konva-Easings.html)
