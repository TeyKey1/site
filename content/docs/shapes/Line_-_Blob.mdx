To create a blob with `Konva`, we can instantiate a `Konva.Line()` object with both `closed` and `tension` properties.

For full list of properties and methods, see the [Line API Reference](/api/Konva.Line.html).

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs>
  <TabItem value="Vanilla" default>
```js live vanilla
import Konva from 'konva';

const stage = new Konva.Stage({
  container: 'container',
  width: window.innerWidth,
  height: window.innerHeight
});

const layer = new Konva.Layer();
stage.add(layer);

const blob = new Konva.Line({
  points: [23, 20, 23, 160, 70, 93, 150, 109, 290, 139, 270, 93],
  fill: '#00D2FF',
  stroke: 'black',
  strokeWidth: 5,
  closed: true,
  tension: 0.3
});

layer.add(blob);
```

  <TabItem value="React">
```js live react
import { Stage, Layer, Line } from 'react-konva';

const App = () => {
  return (
    <Stage width={window.innerWidth} height={window.innerHeight}>
      <Layer>
        <Line
          points={[23, 20, 23, 160, 70, 93, 150, 109, 290, 139, 270, 93]}
          fill="#00D2FF"
          stroke="black"
          strokeWidth={5}
          closed
          tension={0.3}
        />
      </Layer>
    </Stage>
  );
};

export default App;
```
